O que é

Sistema SIGAVERDE é uma ferramenta SIGWEB que permite o gerenciamento e controle da limpeza das áreas verdes urbanas, baseando-se no tempo esperado para o crescimento da vegetação. Maiores detalhes sobre o sistema podem ser vistos em Bourscheidt (2018)

O modelo é voltado inicialmente para as áreas verdes urbanas (praças, parques, canteiros, rotatórias, etc.), mas pode ser adaptado para outras situações, como a manutenção dos sistemas viários em geral.

Para ver o sistema funcionando, clique no link abaixo.

Ir para o mapa

O modelo de crescimento

O SIGAVERDE utiliza um modelo de crescimento da vegetação voltado especificamente para a grama. Esse modelo foi desenvolvido utilizando como base outros modelos existentes na literatura. Maiores detalhes sobre o modelo desenvolvido podem ser encontrados no trabalho de Escobar, E. V. (2019). 

O mapa

O mapa apresenta o estagio de crescimento da vegetação para cada região mapeada, que pode ser uma parque, praça, canteiro, bosque, etc.

Os gráficos

Diferentes tipos de gráficos estão disponíveis para diferentes níveis de acesso: séries temporais dos dados históricos para estudo; gráficos de pizza com taxas de acerto do sistema, etc.
