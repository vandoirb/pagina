O projeto

O sistema é parte de um projeto de pesquisa voltado para a gestão/manutenção de áreas verdes urbanas como parques, praças e canteiros. O projeto contou com o apoio do Conselho Nacional de Pesquisa (CNPq) e da Fundação de Amparo à Pesquisa de São Paulo (FAPESP) através dos seguintes projetos:

- Desenvolvimento de Sistema de Informação Geográfica sobre a web para a gestão ambiental, limpeza e paisagismo no ambiente urbano (proc. 423778/2016-1)

- Proposta de modelo de crescimento de vegetação em áreas urbanas utilizando sistemas de informação geográfica (proc. 17/24038-8);
- Desenvolvimento de um modelo de crescimento de vegetação adaptado para áreas urbanas (proc. 18/12428-9)

1

O problema

Como fazer o gerenciamento da limpeza e paisagismo urbano da melhor forma possível, considerando a dinâmica espacial e temporal das áreas verdes presentes em uma determinada região?
2

O modelo

O SIGAVERDE utiliza um modelo de crescimento de vegetação (gramíneas) baseado em diferentes parâmetros hidrológicos e meteorológicos que são usados junto com modelos estatísticos e diferentes aproximações  para obter o comportamento esperado do crescimento/ciclo anual da vegetação. Ele é calibrado especificamente para o caso das gramíneas e voltado para as áreas urbanas, contendo estratégias para o corte/pode regular.

3

O sistema

O sistema possui diferentes níveis de acesso, possibilitando que gestores, executores e cidadãos em geral acompanhem e coordenem o crescimento da vegetação de uma determinada área dentro dos limites urbanos. Apesar de sua aplicação urbana, ele pode ser estendido para outras situações, como para a gestão dos sistemas viários.
